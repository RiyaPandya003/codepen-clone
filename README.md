# CodePen Clone

A browser-based CodePen-like editor built using vanilla JavaScript.  
The project focuses on layout engineering, resizing logic, and sandboxed code execution.

#Project-link

codepen-clone-html-css-js.netlify.app

## Features

- Live HTML, CSS, and JavaScript preview
- Resizable editor panels (HTML ↔ CSS ↔ JS)
- Vertical resize between editors and preview
- Collapsible panels (CodePen-style behavior)
- Sandboxed execution using iframe
- Real-time console output (log, warn, error)
- Persistent code and layout using LocalStorage

## Tech Stack

- HTML
- CSS
- JavaScript (ES6)
- iframe
- LocalStorage

## Notes

- The **Save** and **Settings** buttons are intentionally disabled (planned features).
- This project is fully client-side.
- Focus was on understanding layout mechanics and event handling rather than using a framework.
